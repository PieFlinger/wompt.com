This is wompt embedded as an iframe

Notice the transparent background and rounded corners. It only takes only line of html to add this to your site and you can customize the text color with a parameter.

Replace the "room_name" in the code below with any text you want to get your own room name.. i.e. "jsconf/2011"

<iframe src='http://wompt.com/chat/room_name?iframe=1#c=444444'
	style='margin:30px; width:800px; height:500px;'
	allowtransparency="true"></iframe>
